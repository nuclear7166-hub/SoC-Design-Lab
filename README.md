# 📖 About

This repository contains FPGA, SoC, and Embedded System projects developed throughout the AIoT Semiconductor Design Course.

The projects focus on digital logic design, FPGA implementation, custom AXI peripherals, embedded software development, and hardware/software co-design using Xilinx tools.

The purpose of this repository is to demonstrate engineering skills and document the learning process through practical projects.
